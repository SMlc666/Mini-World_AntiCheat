# Mini-World_AntiCheat
这是一个基于Lua的迷你世界反作弊项目。该项目旨在提供一系列功能，以防止玩家在游戏中进行作弊行为，从而维护游戏的公平性和健康环境。

## 功能介绍

- **Anti fly**
  - 防止玩家在游戏中进行飞行作弊。

- **Anti Reach**
  - 限制玩家能够触碰远距离方块

- **Anti highJump**
  - 限制玩家的跳跃高度，防止玩家通过作弊方式实现过高跳跃。

- **Lua实现**
  - 所有功能均使用Lua语言编写，方便玩家根据自己的需求进行修改和扩展。
